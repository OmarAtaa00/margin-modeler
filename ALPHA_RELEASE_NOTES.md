# Margin Modeler 0.2.0-alpha.1 — Draft Release Notes

## Alpha notice

This is a prerelease build intended for testing and evaluation. It is not yet a
production-stable release. Keep independent backups of important workspace
exports.

## Highlights

- Native local workspace persistence
- Primary and backup workspace stores
- Migration support for older local storage
- UTC-safe date calculations
- Direct-hours-based resource calculations
- Resource cost and billable totals
- Improved workspace import validation
- Save-queue protection during application close
- Ongoing resource cloning and base-scenario comparison improvements
